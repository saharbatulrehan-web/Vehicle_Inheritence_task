Project Description

This project demonstrates the concept of Inheritance in Object-Oriented Programming (OOP) using Java.

The program models a real-world scenario of vehicles using a hierarchical class structure.

--> OOP Concept Used: Inheritance

The class hierarchy is as follows:

Vehicle (Super Class)
   ↓
Car (Sub Class)
   ↓
ElectricCar (Sub Class)
 Class Details
>> Vehicle (Super Class)
Attributes: brand, speed
Methods:
start()
stop()
display()
toString()
>> Car (Subclass of Vehicle)
Additional Attributes:
numberOfDoors
fuelType
Overrides:
display()
toString()
>> ElectricCar (Subclass of Car)
Additional Attributes:
batteryCapacity
chargingTime
Overrides:
display()
toString()
>> Functionality
Demonstrates constructor chaining using super
Shows method overriding
Displays object information using display() and toString()
Simulates behavior of different vehicle types
